# Portfolio-Thumbnail-Generator
## Overview
An investigation into Unreal Engines UMG (Unreal Motion Graphics). Intention was to create a portfolio thumbnail generator that could be used to create portfolio thumbnails in order to maintain style and structure. Initially, I started creating this as in-game UI before moving it to an **Editor Utility**. This allowed for a much more suitable tool that could be used to create the thumbnail and export it out as a .png
